# Showroom Velocidade 🏎️

**A high-performance WaaS (Website as a Service) platform for car dealerships, optimized for 100/100 Lighthouse performance.**

## 📌 Vision

Showroom Velocidade was built to solve the slow-loading problem of traditional car dealership sites. By using Next.js Server Components and advanced image optimization, we achieve sub-second load times even with high-resolution car galleries.

### Key Features
- **Mobile-First Design**: Optimized for lead conversion on smartphones.
- **WhatsApp Integration**: Instant pre-approval and contact flow.
- **Dynamic Catalog**: Efficient rendering of large inventory lists.
